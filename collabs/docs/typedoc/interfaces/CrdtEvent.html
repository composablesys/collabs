Interface CrdtEvent

Supertype for events emitted by Crdts.

Such events are (typically) emitted after the Crdt processes a local or remote operation.

Some special events (e.g. in CMountPoint) are only associated to a local operation instead of a replicated one; in that case, they should still extend CrdtEvent, and they should always set meta: CrdtEventMeta.local(runtime).

See CrdtEventsRecord.
