Class DefaultElementSerializer<T>

Default serializer. string, number, boolean, undefined, and null types are passed by-value. Crdt types are sent by-reference, using the Crdt's rootId and pathToRoot to identify different replicas of the same Crdt. Other types are passed by-value using BSON (via https://github.com/mongodb/js-bson).
