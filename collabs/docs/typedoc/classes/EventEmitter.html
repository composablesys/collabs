Class EventEmitter<Events>

Classes extending EventEmitter can emit events, and listeners can await events from EventEmitters.

remarks

Inspired by nanoevents, but refactored as an abstract class to better fit into our class hierarchy.

Type parameters

  • Events: EventsRecord

    type of the interface mapping event names to event types.
